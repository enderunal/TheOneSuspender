How to use keyboard shortcuts:

  • Click the "Change" button next to any shortcut to configure it
  • This will open Chrome's extensions keyboard shortcuts page
  • Find "The One Suspender" in the list and set your preferred shortcuts
  • Shortcuts typically use combinations like Ctrl+Shift+Letter
  • Some shortcuts may not have default keys assigned - you can set them manually
